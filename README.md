# Retail-Manager

Ứng dụng quản lý cửa hàng dành cho nhân viên/quản lý cửa hàng của dự án Unified Retail. \
Ứng dụng Expo (React Native) khởi tạo bằng `create-expo-app` và dùng Tamagui cho UI.

## Mô tả ngắn

Ứng dụng dùng để quản lý hàng hoá, tồn kho và xử lý đơn tại quầy.

## Yêu cầu

- Node.js
- - Đã cài đặt và khởi chạy Retail-Backend

## Cài đặt

1. Cài dependencies:

```
npm install
```

2. Tạo file cấu hình môi trường:

```
cp .env.example .env
```

Sau đó chỉnh file .env theo ví dụ

3. Khởi chạy Expo:

```
npx expo start
```

4. Mở trên thiết bị thật bằng Expo Go hoặc trên emulator (Android/iOS).
